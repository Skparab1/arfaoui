How to use

1. Modify the list by hovering over an element and using the "left/right" buttons to add elements and "delete" button to remove elements.
2. Drag blocks of code from the code bank to the current code.
3. Click the digits in blue to change the list contents and function inputs. Select nodes as inputs when prompted.
4. Press "run all" to run all the code instantly.
5. Press "slowly run all" to run the code with delays between each block.
6. Press "step" to run one one block of code at a time.
